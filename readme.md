### v1.5


<h3 id="1234567890v1.5">

<p>6.更改漫画观看页面历史记录追踪逻辑加载的优先级，优化代码</p>

<p>7.漫画观看页面添加正在加载的提示</p>

<p>8.漫画目录页面横屏和竖屏两套不同的布局</p>

<p>9.添加漫画更新推送服务，虽然知道这样做会对后端带来很大的压力，但还是这么做了，此服务一天只能运行一次</p>
